FEATURED GAME

Library Arena

Library Arena is a 2D wave-based survival arena game set inside a trapped library environment. The player must survive increasingly difficult waves of enemies while managing combat, health, and positioning. Each wave introduces stronger enemy types, culminating in a final boss battle at the last wave. The game also includes an Endless Mode and support items such as healing and defence-boosting books, increasing both strategy and replayability.

ASYLUM HORROR (3D)

Asylum Escape

Asylum Escape is a first-person survival horror game set inside a dark, abandoned asylum. The player is trapped inside a terrifying facility where a hostile monster roams the halls, creating constant tension and danger.

The main objective is to escape the asylum by exploring different rooms and searching for the correct exit key. However, the challenge is that multiple fake keys are scattered throughout the map, making it difficult to identify which one is real. Only one of the six randomly spawned keys is the correct escape key.

Gameplay focuses on tension, exploration, and survival. Players must carefully move through the asylum, avoid the monster, and search each room methodically. The keys emit sound effects to help with locating them, but danger increases as the monster reacts to the player’s movement.

The game also includes a volume control system in the main menu for adjusting music and sound effects. Settings can only be changed from the main menu screen.

There are known bugs such as occasional monster pathing issues, but the game represents my first 3D horror project.

Note: This game is only playable on PC or laptop. Download works only on PC/laptop devices.
